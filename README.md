# Chromatic Valley

A relaxing color-matching game built with React Native and Expo. Test your color perception through two challenge types while building a castle as you progress.

## Features

- **Unified Gameplay**: Seamlessly alternates between color matching and color wheel challenges
- **Progressive Castle**: Watch your castle grow as you score points
- **Contextual Tutorials**: Learn mechanics naturally as you encounter them
- **Streak Celebrations**: Duolingo-style celebrations for maintaining streaks
- **Zen Mode**: Practice without timers or pressure
- **Haptic Feedback**: Satisfying tactile responses on supported devices

## Getting Started

### Prerequisites

- Node.js 20.x or higher
- Expo Go app on your mobile device

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npx expo start --clear
```

Scan the QR code with Expo Go (iOS) or the Expo app (Android).

## Tech Stack

- React Native with Expo SDK 54
- TypeScript
- React Native Reanimated for animations
- AsyncStorage for persistence

## Game Modes

### Play Mode
Race against the timer to match colors and complete color wheels. Build your streak for bonus points and watch your castle grow!

### Zen Mode
Take your time with no timer or lives. Perfect for relaxing or practicing color recognition.
